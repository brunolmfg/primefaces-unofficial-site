Workaround for widgetVar undefined

The feature 5850 changed the scope of widgetVar's, removing them from window to new function PF('widgetVar').

// instead
myWigetVar.samplemethod();

// use
PF('myWidgetVar').samplemethod();

// or
var myWidgetVar = PF('myWidgetVar');
myWigetVar.samplemethod();
