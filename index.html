Why?

The New Maintanance Policy don't have freely available releases starting with PrimeFaces 3.5.

This work initiative aims to provide freely available repositories with the result of the integration of the changes from original open source repository. All sources came from the original repository under the Apache License 2.0. The sources from this work are all hosted in GitHub. Fork it!

Disclaimers

This unofficial repository does not contain artifacts for themes. Therefore, keep the official repository in your POM.
In the 3.5.9-UR and 3.5.10-UR releases the feature 5850 ended up coming down from upstream version. See the workaround if you are using one of them.
From 3.5.8-UR releases has one bug in AjaxBehaviour.saveState. In the 3.5.13-UR2 release has fixed.
